**Estado:** Activa

---

## Uso de esta biblioteca

Esta biblioteca reúne el copy base aprobado para las campañas principales de 2026.  
La lógica es simple:

- `manta`: texto breve, jerárquico y de lectura rápida
- `facebook`: texto principal de publicación
- `story`: texto corto, visual y contundente
- `whatsapp`: texto funcional para grupos, difusión o seguimiento

Regla editorial:

- mantener tono parroquial
- evitar texto demasiado largo
- conservar un llamado concreto
- cerrar con una invitación clara
